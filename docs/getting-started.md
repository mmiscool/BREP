# Getting Started

Prerequisites: Node.js 18 or newer and `pnpm` installed.

- Install dependencies: `pnpm install`
- Run the Vite dev server: `pnpm dev`
  - Open the printed URL (usually http://localhost:5173). Try `index.html`, `sdf.html`, or `offsetSurfaceMeshTest.html` for sandbox scenes.
- Run tests: `pnpm test`
- Live testing while editing (Node): `pnpm liveTesting`
